# IMusic - 在线音乐播放系统

一个基于 React + Spring Boot 的在线音乐播放系统，支持歌曲搜索、在线播放、用户登录和收藏管理等功能。

## 技术栈

- **前端**：React 19 + Vite
- **后端**：Spring Boot 3 + Spring Data JPA + Spring Security
- **数据库**：H2 Database

## 快速开始

### 后端启动

```bash
cd imusic-backend
mvn spring-boot:run
```

### 前端启动

```bash
cd imusic-frontend
npm install
npm run dev
```
